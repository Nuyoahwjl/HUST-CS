华中科技大学《计算机通信与网络》课程资料库！🎉


## 仓库内容 📂

本仓库主要包含以下几个部分：

*   **`/课件`** 📚: 包含课程所有章节的PPT课件。
*   **`/作业`** ✍️: 课程相关的作业题目及参考解答。
*   **`/实验`** 🔬: 三次课程实验的详细报告和代码实现。
    *   **`实验一：可靠数据传输`**: 实现了基于 GBN (Go-Back-N) 和 SR (Selective Repeat) 协议的可靠数据传输机制。
    *   **`实验二：Socket编程`**: 通过 Socket API 实现了客户端/服务器模型，涵盖了网络应用层编程的基础。
    *   **`实验三：SDN实验`**: 基于 Mininet 和 OpenFlow 的软件定义网络实验，探索了现代网络架构的控制与转发分离思想。
