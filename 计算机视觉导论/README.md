# 计算机视觉导论

该目录存放了华中科技大学《计算机视觉导论》课程的相关资料，包括课件、实验、作业和课程项目。

## 目录结构

```
.
├── courseware/         # 课程课件
├── exp-1/              # 实验一
├── exp-2/              # 实验二
├── exp-3/              # 实验三
├── homework/           # 课后作业
└── super-resolution/   # 课程大作业：图像超分辨率
```

## 内容说明

### 课件 ([courseware](courseware/))

包含了课程教学的全部PPT和PDF文件。

- `1-绪论.pdf`
- `2-图像处理基础.pdf`
- `3-边缘检测、线检测.pdf`
- `4-特征检测与匹配.pdf`
- `5-前馈神经网络.pdf`
- `6-卷积神经网络.pptx`
- `7-循环神经网络.pptx`

### 实验 ([exp-1](exp-1/), [exp-2](exp-2/), [exp-3](exp-3/))

每个实验文件夹包含实验指导书、Jupyter Notebook 格式的源代码、实验报告以及所需的数据集。

- **实验一**: 前馈神经网络实现。
- **实验二**: 基于MNIST数据集的卷积神经网络实践。
- **实验三**: 神经网络权重剪枝。

### 作业 ([homework](homework/))

包含课程作业题目 (`hw.docx`) 和参考答案 (`ans.md`, `ans.pdf`)。

### 大作业 ([super-resolution](super-resolution/))

关于图像超分辨率（Super-Resolution）的课程项目，内含项目报告和代码实现。
