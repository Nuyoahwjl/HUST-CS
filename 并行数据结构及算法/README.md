# 📚 并行数据结构及算法

该目录包含了华中科技大学《并行数据结构及算法》课程的学习资料，包括课程课件、实验代码和实验报告。

## 📁 目录结构

```
.
├── courseware/         # 课程课件
├── exp/                # 实验相关
└── README.md           # 说明文档
```

## 📝 内容说明

### 🧑‍🏫 课件 ([courseware](courseware/))

包含了课程的教学幻灯片（PDF）、参考书籍和习题答疑。主要内容包括：

-   Introduction
-   SPARC
-   Algorithmic Analysis
-   Sequences
-   Algorithmic Design
-   Randomized Algorithms
-   Graphs and Graph Search
-   Review

### 🧪 实验 ([exp](exp/))

包含了课程实验的全部内容，包括 SML 语言基础练习、实验报告要求及个人报告。

-   **[SML语言基础](exp/SML语言基础/) & [SML语言进阶](exp/SML语言进阶/)**: ⌨️ Standard ML 语言的编程练习代码。
-   **[25-实验报告要求/](exp/25-实验报告要求/)**: 📄 包含实验报告的封面模板、排版规范和提交流程指南。
-   **[Report/](exp/report/)**: ✍️ 存放已完成的个人实验报告。
